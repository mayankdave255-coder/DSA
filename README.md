# DSA & CP Solutions

A collection of Data Structures and Algorithms problems solved in C++.  
Used for practice, reference, and revision.

## Structure

- Solutions organised by topic and platform.
- Each file includes problem name, approach, and complexity.
- Templates for commonly used data structures.

## Topics

Arrays, Strings, Linked Lists, Stacks, Queues, Recursion, Sorting, Searching, Hashing, Trees, Heaps, Dynamic Programming, Graphs, Tries, Bit Manipulation, Advanced DS, Maths.

## Platforms

LeetCode, HackerEarth, GeeksforGeeks, CodeChef, Codeforces.

## Usage

Browse folders by topic to find implementations and solutions.  
All code is written in standard C++17.
